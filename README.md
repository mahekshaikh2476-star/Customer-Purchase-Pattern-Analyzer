# Customer Purchase Pattern Analyzer

## Overview

Customer Purchase Pattern Analyzer is a data analytics project developed using Python, Pandas, Matplotlib, and Seaborn. The project explores customer purchasing behavior, identifies spending trends, analyzes revenue distribution, and provides business insights through visualizations.

---

## Objectives

- Analyze customer purchase behavior
- Identify top-performing regions and categories
- Understand customer demographics
- Evaluate payment method preferences
- Discover revenue trends and customer segments

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- GitHub

---

## Dataset Details

The dataset contains 100 customer records with the following information:

- Customer ID
- Customer Name
- Age
- Gender
- Region
- Purchase Category
- Purchase Amount
- Loyalty Status
- Payment Method
- Purchase Date
- Last Purchase Date

## Key Insights

- Revenue varies significantly across regions.
- Certain product categories generate higher sales.
- Customer age influences purchasing behavior.
- Loyalty program members contribute substantial revenue.
